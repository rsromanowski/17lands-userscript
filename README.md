# 17Lands Userscript Filter

## Usage
### Card Data
1. Press `/` to focus on input
2. Type partial card names to filter table. Examples: `Candy Grapple` or `torch`
3. Separate by comma to see multiple cards. Example: `candy,torch,cooped`
